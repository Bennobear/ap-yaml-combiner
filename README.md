# Archipelago YAML Combiner

A small GUI tool that lets you combine multiple Archipelago player YAML files into one.  
It supports drag & drop, weights, and automatic trigger generation.

## Features

- Combine multiple Archipelago YAML files
- Assign a weight to each YAML
- Automatically generate proper `triggers:` entries
- Drag & drop YAML files into the interface
- Simple, clean Tkinter-based UI

## Installation

Install required packages:

    pip install -r requirements.txt

Run the program:

    python ap_yaml_combiner.py

## Files in this repository

- `ap_yaml_combiner.py` — the main application
- `requirements.txt` — Python dependencies
- `.gitignore` — ignores Python caches and build artifacts
